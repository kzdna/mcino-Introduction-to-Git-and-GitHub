# Simple Interest Calculator

A simple Python script to calculate interest based on the principal amount, annual rate of interest, and time period in years.

## Description
This project is part of the IBM "Introduction to Git and GitHub" course. It serves as a practical example to demonstrate basic Git workflows, including branching, merging, and pull requests.

## Formula
The simple interest is calculated using the following formula:
**A = P(1 + rt)**

Where:
- **A** = Total amount (Principal + Interest)
- **P** = Principal amount
- **r** = Annual interest rate (decimal)
- **t** = Time period in years

## How to Use
1. Clone the repository to your local machine.
2. Run the script (if applicable) or use the logic to calculate interest.
3. Input the required values for Principal, Rate, and Time.
